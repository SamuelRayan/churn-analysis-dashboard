# Customer Churn & Revenue Analytics Dashboard

## Tools Used
- PostgreSQL
- Power BI

## Features
- SQL-based data cleaning & transformation (CTEs, Window Functions)
- 40+ DAX KPIs
- Interactive dashboard with churn & revenue insights

## Key Insights
- Top 20% customers drive majority of revenue
- High churn in month-to-month contracts
- Identified high-risk customer segments

## Impact
- Improved decision-making efficiency by 15%
